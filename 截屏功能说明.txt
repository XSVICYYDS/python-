"""
截屏功能使用说明
小白桌面宠物的矩形区域截屏功能
"""

功能说明：
==============

优化后的截屏功能现在支持矩形区域选择截屏，而不是之前的全屏截图。

主要特性：
-----------

1. **矩形区域选择**
   - 用户可以通过鼠标左键拖动选择任意矩形区域
   - 选区会实时显示尺寸信息（宽度 × 高度）
   - 支持从任意角落或边缘开始选择

2. **可视化选区**
   - 蓝色边框标识选区范围
   - 四角有明显的标记点
   - 选区中心下方显示尺寸信息

3. **灵活的交互**
   - 左键拖动：选择截取区域
   - 右键点击：取消截图
   - ESC键：取消截图
   - Enter键：确认选区并保存
   - 选择完成后自动弹出保存对话框

4. **智能边界处理**
   - 选区自动限制在屏幕范围内
   - 尺寸标签自动调整位置避免超出屏幕
   - 最小选区限制（5×5像素）

5. **宠物窗口保护**
   - 截图时自动隐藏宠物窗口
   - 避免截图包含宠物自身
   - 截图完成后自动恢复宠物显示

使用流程：
-----------

1. 在小白菜单中选择"工具" -> "截屏"
2. 宠物窗口会自动隐藏
3. 屏幕会出现半透明遮罩
4. 使用鼠标左键拖动选择要截取的区域
5. 选择完成后会弹出保存对话框
6. 选择保存路径和文件名，点击"保存"
7. 截图自动保存，宠物窗口恢复显示

技术实现：
-----------

- 使用PyQt5的QWidget作为截屏窗口
- 使用QPixmap.grabWindow()捕获全屏
- 使用QPainter绘制半透明遮罩和选区
- 使用QPainter.CompositionMode实现选区透明效果
- 非模态窗口设计，不阻塞用户操作

文件结构：
-----------

- screen_capture.py：截屏功能实现
- main.py：集成截屏功能到宠物应用
- test_screen_capture.py：独立测试脚本

测试方法：
-----------

运行测试脚本查看截屏功能：
```bash
python test_screen_capture.py
```

或在小白应用中使用：
1. 右键点击小白
2. 选择"工具" -> "截屏"
3. 按提示操作

常见问题：
-----------

Q: 为什么截图时宠物窗口会自动隐藏？
A: 这是为了避免截图包含宠物自身，如果需要截取包含宠物的画面，可以先截图后再显示宠物。

Q: 可以截取超出屏幕范围的区域吗？
A: 不能，截屏功能仅支持截取当前屏幕可见区域。

Q: 支持截取多屏幕吗？
A: 当前版本仅支持主屏幕截取，不支持多屏幕合并。

Q: 为什么选区太小无法保存？
A: 系统要求选区至少为5×5像素，太小的选区会被忽略。

Q: 截图保存在哪里？
A: 默认保存在用户图片文件夹（Windows为Pictures文件夹），也可以在保存对话框中选择其他位置。
