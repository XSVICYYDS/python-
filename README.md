## 📈 智能推荐算法

```python
推荐分数 = 使用次数×10 + 时间段权重×5 + 近期使用权重×1

特点:
- 使用越频繁,推荐优先级越高
- 当前时段常用的功能优先推荐
- 最近7天使用的功能加权推荐
- 综合评分排序后选择Top N
```

## 💾 数据存储

### 存储格式
```json
{
    "selections": {"功能ID": true/false},
    "usage": {
        "功能ID": {
            "count": 使用次数,
            "last_used": ISO时间,
            "hours": {"小时": 次数}
        }
    },
    "last_updated": ISO时间
}
```

### 存储位置
- **Windows**: `%APPDATA%\MalteseDesktopPet\selection_history.json`
- **macOS**: `~/Library/Application Support/MalteseDesktopPet/selection_history.json`
- **Linux**: `~/.maltese_desktop_pet/selection_history.json`
