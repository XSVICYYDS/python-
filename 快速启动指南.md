# 🎯 功能选择列表 - 快速启动指南

## 🚀 快速开始

### 1. 运行测试程序

双击运行 `test_feature_list.py` 查看组件效果

```bash
cd 小白-源代码
python test_feature_list.py
```

### 2. 在配置中使用

打开小白桌面宠物 → 右键菜单 → 配置 → 切换到"功能选择"标签页

## 💡 基本操作

### 选择功能

1. **单个选择**: 点击功能项或勾选复选框
2. **全选**: 点击"全选"按钮
3. **取消全选**: 点击"取消"按钮
4. **智能推荐**: 点击"🤖 智能推荐"按钮

### 保存配置

1. 在"功能选择"标签页完成选择
2. 点击"💾 保存选择"按钮
3. 点击"确定"应用所有配置

self.feature_list = FeatureSelectionList(self.base_dir, self.config)

## 🐛 故障排除

### 问题1: 组件无法显示
**解决方案:**
- 检查PyQt5是否正确安装
- 检查feature_list_component.py是否存在
- 查看控制台错误信息

### 问题2: 智能推荐无反应
**解决方案:**
- 可能是首次使用,没有历史数据
- 点击"全选"手动选择功能
- 查看日志文件debug.log

### 问题3: 保存失败
**解决方案:**
- 检查数据目录是否有写入权限
- 确认磁盘空间充足
- 查看错误提示信息
