# 小白桌面宠物打包指南

## 环境要求

1. **Python 3.11+**
2. **PyInstaller 6.x+**
3. **Inno Setup 6.x+**
4. **PyQt5**
5. **Pillow**

## 依赖安装

首先安装所有必要的依赖：

```bash
# 安装PyInstaller
pip install pyinstaller

# 安装PyQt5
pip install PyQt5

# 安装Pillow（用于屏幕截图）
pip install Pillow

# 安装win10toast（用于Windows通知）
pip install win10toast

# 安装plyer（跨平台通知库）
pip install plyer

# 安装所有依赖
pip install -r requirements.txt
```

## 打包步骤

### 1. 准备工作

确保项目目录结构正确：

```
小白-源代码/
├── main.py              # 主入口文件
├── GIF/                  # 动画文件夹
├── Image/                # 图片资源文件夹
├── games/               # 游戏模块
├── 智能桌面宠物-小白.spec  # PyInstaller配置文件
└── requirements.txt      # 依赖列表
```

### 2. 创建requirements.txt

确保 `requirements.txt` 包含所有依赖：

```
PyQt5==5.15.10
Pillow==10.2.0
win10toast==0.9
plyer==2.1.0
```

### 3. 使用PyInstaller打包

#### 方法一：使用spec文件（推荐）

```bash
# 进入源代码目录
cd 小白-源代码

# 运行PyInstaller
pyinstaller 小白.spec
```

#### 方法二：命令行打包

```bash
# 基本打包命令
pyinstaller --name="智能桌面宠物-小白" \
            --onefile \
            --noconsole \
            --icon=Image/MenuIcon.jpg \
            --add-data="GIF;GIF" \
            --add-data="Image;Image" \
            --hidden-import=PyQt5 \
            --hidden-import=PIL \
            --hidden-import=PIL.ImageGrab \
            --hidden-import=win10toast \
            --hidden-import=plyer \
            main.py
```

### 4. 打包后的文件位置

打包完成后，可执行文件会在：

```
小白-源代码/dist/智能桌面宠物-小白/
├── 智能桌面宠物-小白.exe  # 主程序
├── GIF/                   # 动画文件
├── Image/                 # 图片资源
└── python3x.dll          # Python运行时库
```

### 5. 使用Inno Setup创建安装程序

#### 5.1 安装Inno Setup

下载并安装 Inno Setup 6.x：https://jrsoftware.org/isinfo.php

#### 5.2 配置安装程序

编辑 `installer.iss` 文件，根据实际情况修改路径：

```iss
; 修改源路径
Source: "..\dist\智能桌面宠物-小白\*"; DestDir: "{app}"; Flags: ignoreversion recursesubdirs createallsubdirs

; 修改图标（如果需要）
SetupIconFile=Image\MenuIcon.ico
```

#### 5.3 创建图标（可选）

如果需要 ICO 格式的图标，可以使用在线工具将 JPG 转换为 ICO：
- https://www.icoconverter.com/
- https://convertio.co/jpg-ico/

#### 5.4 编译安装程序

双击 `installer.iss` 文件，或使用命令行：

```bash
# 使用Inno Setup编译器
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

#### 5.5 安装程序输出

编译完成后，安装程序会在 `小白-源代码/安装包/` 目录生成：

```
智能桌面宠物-小白-安装包.exe
```

## 常见问题

### 1. 打包后程序无法运行

**问题**：打包后的程序启动失败或闪退。

**解决方案**：
- 检查是否缺少必要的 DLL 文件
- 使用 `--debug` 模式打包查看错误信息
- 确保所有依赖项都包含在 `hidden-imports` 中

### 2. 图片资源无法加载

**问题**：程序运行时无法找到 GIF 或图片文件。

**解决方案**：
- 检查 `--add-data` 参数是否正确
- 确保 `*.spec` 文件中的 `datas` 路径正确
- 使用绝对路径测试

### 3. 控制台窗口闪烁

**问题**：程序启动时显示控制台窗口。

**解决方案**：
- 使用 `--noconsole` 参数
- 在 spec 文件中设置 `console=False`

### 4. 权限问题

**问题**：程序需要管理员权限。

**解决方案**：
- 在 spec 文件中设置 `uac_admin=False`
- 或在 Inno Setup 中设置 `PrivilegesRequired=lowest`

## 高级配置

### 1. 自定义图标

如果需要自定义图标：

1. 准备 256x256 像素的 PNG 图片
2. 使用在线工具转换为 ICO 格式
3. 在 spec 文件中设置：
   ```python
   exe = EXE(..., icon='Image/MenuIcon.ico', ...)
   ```

### 2. 版本信息

在 spec 文件中设置版本信息：

```python
version_info = {
    'version': '1.0.0',
    'company_name': '尚志中学809班',
    'file_description': '智能桌面宠物-小白',
    'product_name': '智能桌面宠物-小白',
}
```

### 3. 单文件打包

如果想打包为单个 EXE 文件：

```bash
pyinstaller --onefile --noconsole --name="智能桌面宠物-小白" main.py
```

注意：单文件打包会导致启动较慢，且需要解压临时文件。

## 优化建议

1. **排除不需要的模块**：减少打包体积
2. **使用 UPX 压缩**：进一步减小文件大小
3. **分离依赖**：将运行时库单独打包
4. **代码签名**：为程序添加数字签名（提高安全性）

## 技术支持

如果在打包过程中遇到问题，请检查：

1. Python 版本是否兼容
2. 所有依赖是否正确安装
3. 路径是否正确（注意 Windows 路径分隔符）
4. 是否有权限问题
