# 功能选择列表组件使用说明

## 概述

功能选择列表是一个完整的交互式UI组件,用于在小白配置中展示和管理可选择的工具/功能项。

## 核心功能

### 1. 交互式选择机制

- **点击选择**: 单击功能项可以切换选中状态
- **复选框选择**: 使用复选框精确控制选择状态
- **全选功能**: 一键选择所有功能
- **取消全选**: 一键取消所有选择
- **实时反馈**: 选中状态实时显示和更新

### 2. 智能推荐系统

点击"🤖 智能推荐"按钮,系统会根据以下因素自动选择功能:

- **使用频率**: 根据历史使用次数推荐
- **使用时间模式**: 分析不同时间段的使用习惯
- **最近使用**: 优先推荐近期使用过的功能
- **综合评分**: 计算综合得分并推荐高分功能

### 3. 动画效果

- **点击动画**: 功能项点击时的缩放动画效果
- **悬停效果**: 鼠标悬停时的阴影和边框变化
- **平滑过渡**: 选中状态切换时的平滑过渡效果

### 4. 数据持久化

- **自动保存**: 选择状态自动保存到本地
- **历史记录**: 记录功能使用历史
- **跨会话**: 重新启动后自动恢复上次选择

## 使用方法

### 在配置对话框中使用

1. 打开小白配置对话框
2. 切换到"功能选择"标签页
3. 浏览和选择需要的功能
4. 点击"保存选择"按钮保存
5. 点击"确定"应用所有配置

### 智能推荐流程

1. 点击"🤖 智能推荐"按钮
2. 系统分析使用习惯
3. 显示推荐数量和依据
4. 自动选择推荐的功能

## 技术实现

### 核心类

- **FeatureItem**: 单个功能项组件
- **FeatureSelectionList**: 功能选择列表主组件
- **SelectionManager**: 选择状态管理器
- **RecommendationEngine**: 智能推荐引擎

### 数据结构

```python
{
    "selections": {
        "stick": true,
        "call": false,
        ...
    },
    "usage": {
        "stick": {
            "count": 10,
            "last_used": "2024-01-01T12:00:00",
            "hours": {"9": 5, "14": 3, "18": 2}
        },
        ...
    }
}
```

### 存储位置

- Windows: `%APPDATA%\MalteseDesktopPet\selection_history.json`
- macOS: `~/Library/Application Support/MalteseDesktopPet/selection_history.json`
- Linux: `~/.maltese_desktop_pet/selection_history.json`

## 性能优化

- 使用Qt的GraphicsDropShadowEffect优化阴影渲染
- 动画使用QPropertyAnimation实现60fps流畅动画
- 按需加载功能项,减少内存占用
- 异步保存数据,不影响UI响应

## 扩展性

组件设计遵循开闭原则,易于扩展:

- 轻松添加新的功能分类
- 支持自定义功能项图标
- 可配置推荐算法参数
- 支持多种数据存储后端

## 错误处理

- 文件读取失败时使用默认配置
- 数据格式错误时自动恢复
- UI异常时显示友好提示
- 推荐失败时提供默认推荐
