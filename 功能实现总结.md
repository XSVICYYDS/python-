## 🎯 核心算法

### 推荐评分算法

```python
score = 使用次数×10 + 时间段权重×5 + 近期使用权重×1

其中:
- 使用次数: 历史使用总次数
- 时间段权重: 当前小时的累计使用次数
- 近期使用权重: (7-天数)×3,最大30
```

### 选择状态管理

```python
状态机:
UNSELECTED ←→ SELECTED
   ↓
   └── click/toggle
```

## 🚀 使用方式

### 基本使用

```python
from feature_list_component import FeatureSelectionList
from config import Config

# 创建配置和组件
config = Config(base_dir)
feature_list = FeatureSelectionList(base_dir, config)

# 监听选择变化
feature_list.selectionChanged.connect(on_selection_changed)

# 获取选择状态
selections = feature_list.getSelections()

# 全选/取消全选
feature_list.selectAll()
feature_list.deselectAll()

# 智能推荐
feature_list.applySmartRecommendation()

# 保存选择
feature_list.saveSelections()
```

### 在配置对话框中使用

1. 打开配置对话框
2. 切换到"功能选择"标签
3. 进行选择和配置
4. 保存并应用
