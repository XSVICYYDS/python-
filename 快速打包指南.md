# PyInstaller + Inno Setup 快速打包指南

## 快速开始

### 方法一：使用自动化脚本（推荐）

```bash
# 1. 进入源代码目录
cd "小白-源代码"

# 2. 运行自动化打包脚本
python 打包脚本.py
```

脚本会自动完成：
- ✅ 检查依赖
- ✅ 清理旧文件
- ✅ PyInstaller打包
- ✅ 测试打包结果
- ✅ 创建安装程序（可选）

---

### 方法二：手动打包

#### 第1步：安装依赖

```bash
pip install pyinstaller PyQt5 Pillow win10toast plyer
```

#### 第2步：PyInstaller打包

```bash
# 进入源代码目录
cd "小白-源代码"

# 运行PyInstaller
pyinstaller 小白.spec
```

或使用命令行：

```bash
pyinstaller --name="智能桌面宠物-小白" \
            --onefile \
            --noconsole \
            --icon=Image/MenuIcon.jpg \
            --add-data="GIF;GIF" \
            --add-data="Image;Image" \
            --hidden-import=PyQt5 \
            --hidden-import=PIL \
            --hidden-import=win10toast \
            --hidden-import=plyer \
            main.py
```

#### 第3步：创建安装程序

安装 Inno Setup 6（如果未安装）：
https://jrsoftware.org/isinfo.php

编译安装程序：

```bash
# 方法1：双击 installer.iss
# 方法2：命令行
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

## 常见问题

### Q1: 打包后程序无法运行？
```bash
# 使用debug模式打包
pyinstaller --debug main.py
```

### Q2: 缺少DLL文件？
确保在spec文件中包含所有依赖：
```python
hiddenimports=[
    'PyQt5',
    'PIL',
    'win10toast',
]
```

### Q3: 图标不显示？
将JPG转换为ICO格式：
- 在线工具：https://www.icoconverter.com/
- 或修改spec文件：`icon=None`

### Q4: 控制台窗口闪烁？
确保使用 `--noconsole` 参数：
```python
exe = EXE(..., console=False, ...)
```

## 高级配置

### 修改安装程序信息

编辑 `installer.iss`：

```iss
AppName=你的应用名称
AppVersion=1.0.0
AppPublisher=你的名字
```

### 添加额外的文件

编辑 `小白.spec`：

```python
datas=[
    ('GIF', 'GIF'),
    ('Image', 'Image'),
    ('额外文件夹', '目标文件夹'),
]
```

### 单文件打包

修改 `小白.spec`，添加 `--onefile`：

```python
exe = EXE(
    pyz,
    a.scripts,
    [],
    exclude_binaries=True,  # 改为True
    ...
)
```

## 卸载程序

用户可以通过以下方式卸载：
1. 控制面板 → 程序和功能
2. 开始菜单 → 程序 → 卸载
3. 运行安装程序选择卸载

卸载时会自动：
- ✅ 删除所有安装文件
- ✅ 删除快捷方式
- ✅ 移除开机自启动注册表项
- ✅ 保留用户数据（可选）
