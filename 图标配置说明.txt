## 注意事项

### 1. 图标文件要求
- 格式：`.ico` 格式
- 建议尺寸：256x256 像素
- 如果不是 ICO 格式，请先转换：
  - 在线工具：https://www.icoconverter.com/
  - 或使用 Python 代码转换：
    ```python
    from PIL import Image
    img = Image.open('小白.png')
    img.save('小白.ico', format='ICO')
    ```

### 2. 图标文件位置
图标文件应放置在以下位置之一：
- **绝对路径（推荐）**: `C:\Users\XS\Desktop\小白.ico`
- **项目目录**: 复制到 `小白-源代码/Image/` 目录下

### 3. 复制图标到项目目录（可选）

如果你想将图标复制到项目目录以便分发：

```bash
copy "C:\Users\XS\Desktop\小白.ico" "小白-源代码\Image\"
```

然后更新配置文件中的路径：
```python
# 小白.spec
icon='Image/小白.ico'

# installer.iss
SetupIconFile=Image\小白.ico
```

## 常见问题

### Q1: 图标文件不存在？
**错误**: `Icon not found`
**解决**: 确认图标文件路径是否正确

### Q2: 图标格式错误？
**错误**: `Cannot load icon`
**解决**: 将图片转换为 ICO 格式

### Q3: 图标不显示？
**原因**: Windows缓存问题
**解决**: 重启电脑或清理图标缓存
